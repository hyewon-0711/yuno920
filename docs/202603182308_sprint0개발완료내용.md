## 1. 디자인 시스템 코드화

### 주요 변경: Tailwind CSS 제거 → CSS Modules + CSS Variables

**제거한 패키지:**
- `tailwindcss`
- `@tailwindcss/postcss`

**대체 방식:**
- 순수 CSS Variables (`globals.css`)로 디자인 토큰 정의

## 3. Supabase Auth 설정

### 프론트엔드 인증 코드

| 파일 | 역할 |
|------|------|
| `lib/supabase.ts` | Supabase 클라이언트 초기화 |
| `lib/auth.ts` | 인증 유틸리티 (signUp, signIn, signInWithOAuth, signOut, getSession, getUser, onAuthStateChange) |
| `lib/api.ts` | FastAPI 백엔드 API 클라이언트 (GET/POST/PUT/DELETE + Bearer 토큰) |
| `contexts/AuthContext.tsx` | React 인증 컨텍스트 (AuthProvider, useAuth 훅) |
| `app/auth/callback/route.ts` | OAuth 콜백 처리 (code → session 교환) |

### 지원 인증 방식

- 이메일/비밀번호 로그인 및 회원가입
- OAuth (Google, Kakao) — Supabase 대시보드에서 프로바이더 활성화 필요

│   └── callback/route.ts        # OAuth 콜백

├── ui/                           # 공통 UI 컴포넌트 (9개)
