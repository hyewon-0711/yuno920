# Yuno920 사용자 흐름도 (User Flow)

---

## 1. 첫 사용 플로우 (온보딩)

```
[앱 진입]
   │
   ▼
[로그인 여부 확인] ──── 로그인됨 ──→ [Home (Today)]
   │
   로그인 안됨
   │
   ▼
[랜딩 페이지]
   │
   ├── "시작하기" ──→ [회원가입]
   │                      │
   │                      ▼
   │                 [이메일/비밀번호 입력]
   │                      │
   │                      ▼
   │                 [아이 프로필 등록]
   │                   - 이름
   │                   - 생년월일
   │                   - 성별
   │                   - 출생 시간 (선택)
   │                   - 사진 (선택)
   │                      │
   │                      ▼
   │                 [Home (Today)] ← 첫 화면
   │
   └── "로그인" ──→ [로그인 페이지]
                       │
                       ├── 이메일 로그인
                       ├── Google 로그인
                       └── 카카오 로그인
                            │
                            ▼
                       [Home (Today)]
```

## 2. 일일 루틴 플로우 (부모 — 핵심 시나리오)

```
[앱 열기]
   │
   ▼
[Home (Today)]
   │
   ├── AI 코칭 확인 (읽기만)
   │
   ├── 오늘 일정 확인
   │     └── [+ 일정 추가] → 제목/시간 입력 → 저장 → 목록 갱신
   │
   ├── 독서 현황 확인
   │     └── 탭하면 → [독서 기록] 상세로 이동
   │
   ├── 급식 메뉴 확인 (읽기만)
   │
   └── "오늘 기록하기" 클릭
         │
         ▼
      [기록 작성 페이지]
         │
         ├── 기분 선택 (😊😐😢🤒😴)
         ├── 카테고리 선택
         ├── 텍스트 입력
         └── 사진 첨부 (선택)
              │
              ▼
           [저장]
              │
              ├── Supabase DB 저장
              ├── 사진 → Supabase Storage 업로드
              └── 자동 태깅 (FastAPI → OpenAI)
                    │
                    ▼
              [타임라인에 반영]
              토스트: "기록이 저장되었습니다 ✅"
```

## 3. 기록 열람 플로우

```
[Record 탭]
   │
   ▼
[기록 목록]
   │
   ├── [타임라인 뷰] ← 기본
   │     - 날짜별 그룹
   │     - 스크롤로 과거 기록 열람
   │
   └── [캘린더 뷰]
         - 달력에서 날짜 클릭
              │
              ▼
         해당 날짜의 기록 표시
   │
   기록 카드 클릭
   │
   ▼
[기록 상세]
   │
   ├── 내용 열람
   ├── [수정] → 수정 폼 → 저장
   └── [삭제] → 확인 팝업 → 삭제 → 목록으로
```

## 4. 독서 기록 플로우

```
[Record 탭] 또는 [Home → 독서 현황]
   │
   ▼
[독서 기록 목록]
   │
   └── [+ 독서 기록]
         │
         ▼
      [독서 기록 작성]
         ├── 책 제목 입력
         ├── 카테고리 선택 (과학/동화/역사 등)
         ├── 읽은 날짜
         ├── 읽은 시간 (분)
         └── 메모 (선택)
              │
              ▼
           [저장]
              │
              ├── DB 저장
              ├── Home 독서 현황 갱신
              ├── Growth 독서 성장 데이터 갱신
              └── Play 독서 퀴즈에 반영 가능
```

## 5. 성장 확인 플로우 (부모)

```
[Growth 탭]
   │
   ▼
[성장 대시보드]
   │
   ├── 신체 성장 카드
   │     ├── 그래프 확인 (읽기)
   │     └── [+ 입력] → 키/몸무게 입력 → 저장 → 그래프 갱신
   │
   ├── 6각형 인재 분석
   │     ├── 레이더 차트 확인 (읽기)
   │     └── AI 분석 코멘트 확인
   │
   ├── 학습 지표
   │     ├── SR 점수 추이 확인
   │     └── [+ 입력] → 점수 입력 → 저장
   │
   └── 독서 성장
         ├── 월별 독서량 차트
         └── 분야별 분포 차트
```

## 6. 놀이 플로우 (아이)

```
[Play 탭]
   │
   ▼
[놀이터 메인]
   │
   ├── [계산 게임] 선택
   │     │
   │     ▼
   │   [게임 시작 화면]
   │     - 현재 레벨 표시
   │     - "시작" 버튼
   │        │
   │        ▼
   │   [게임 플레이]
   │     - 문제 표시
   │     - 답 입력 (터치 키패드)
   │     - 맞으면 다음 문제
   │     - 틀리면 정답 표시
   │        │
   │        ▼ (10문제 또는 시간 종료)
   │   [결과 화면]
   │     - 점수
   │     - 정답률
   │     - 레벨 변화
   │     - "다시하기" / "돌아가기"
   │        │
   │        ▼
   │   play_logs에 저장 → hexagon_scores 갱신
   │
   ├── [기억력 게임] (유사 흐름)
   │
   └── [독서 퀴즈]
         │
         ▼
      [최근 읽은 책 기반 퀴즈]
        - FastAPI → OpenAI로 문제 생성
        - 객관식 3~5문제
        - 결과 저장
```

## 7. AI 리포트 플로우

```
[Insight 탭]
   │
   ▼
[AI 인사이트 메인]
   │
   ├── [하루] 탭 ← 기본
   │     │
   │     ▼
   │   [하루 요약 카드]
   │     - 자동 생성됨 (매일 밤 or 요청 시)
   │     - 없으면: "요약 생성하기" 버튼
   │          │
   │          ▼
   │        FastAPI → Supabase에서 오늘 데이터 조회
   │                → OpenAI로 요약 생성
   │                → ai_reports에 저장
   │                → 화면에 표시
   │
   ├── [주간] 탭
   │     - 주간 행동 패턴 분석
   │     - 변화 포인트 하이라이트
   │
   ├── [월간] 탭
   │     - 성장 분석 리포트
   │     - 독서 성향 분석
   │
   └── [기질 리포트] 카드 클릭
         │
         ▼
      [기질/사주 리포트 상세]
        - 입력: 생년월일 + 출생시간
        - FastAPI → OpenAI로 분석
        - 성향 분석 + 양육 가이드 표시
        - 1회 생성 후 캐싱
```

## 8. 설정 / 프로필 플로우

```
[Home 헤더 → 프로필 아이콘] 또는 [설정 메뉴]
   │
   ▼
[설정 페이지]
   │
   ├── 아이 프로필 편집
   │     └── 이름/생년월일/사진 수정 → 저장
   │
   ├── 가족 관리
   │     ├── 가족 목록 확인
   │     └── [+ 가족 초대]
   │           └── 초대 링크 생성 → 공유
   │                 → 수신자가 링크 클릭 → 가입 → family_members에 추가
   │
   ├── 알림 설정
   │     ├── 기록 리마인더 ON/OFF
   │     ├── 리포트 알림 ON/OFF
   │     └── 알림 시간 설정
   │
   └── 로그아웃
         └── 확인 팝업 → 세션 삭제 → 랜딩 페이지
```

## 핵심 화면 우선순위 정리

### P0 (Sprint 0~1에 구현)

| 화면 | 플로우 |
|------|--------|
| 랜딩/온보딩 | 플로우 1 |
| 로그인/회원가입 | 플로우 1 |
| 아이 프로필 등록 | 플로우 1 |
| Home (Today) | 플로우 2 |
| 기록 작성 | 플로우 2 |
| 기록 목록/타임라인 | 플로우 3 |

### P1 (Sprint 2~3에 구현)

| 화면 | 플로우 |
|------|--------|
| 기록 상세/수정/삭제 | 플로우 3 |
| 독서 기록 | 플로우 4 |
| Growth 대시보드 | 플로우 5 |
| Play 메인 + 게임 | 플로우 6 |

### P2 (Sprint 4~5에 구현)

| 화면 | 플로우 |
|------|--------|
| AI 리포트 | 플로우 7 |
| 기질 리포트 | 플로우 7 |
| 설정/프로필 | 플로우 8 |
| 가족 초대 | 플로우 8 |
