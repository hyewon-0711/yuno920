# 📘 PRD: AI 기반 아이 성장·육아 통합 플랫폼

---

# 1. 제품 개요 (Product Overview)

## 1.1 제품 정의

**아이의 하루, 성장, 독서, 놀이, 기억을 AI 기반으로 통합 관리하는 부모·아이 공동 플랫폼**

---

## 1.2 목표

### 비즈니스 목표

* DAU 중심 Retention 서비스 구축
* AI 기반 구독 모델 확장
* 향후 교육/학원/콘텐츠 B2B 확장

### 사용자 목표

#### 부모

* 아이의 하루를 한눈에 파악
* 성장 상태를 데이터로 확인
* 육아 의사결정을 AI 도움으로 수행

#### 아이

* 재미있는 게임/학습 경험
* 성취감 및 보상 경험

---

## 1.3 핵심 KPI

* D1 / D7 Retention
* DAU / MAU
* 하루 평균 접속 횟수
* 기록 생성 수
* 게임 실행 수
* 독서 기록 수

---

# 2. 사용자 정의 (User Persona)

## 부모 (Primary User)

* 30~40대
* 초등학생 자녀 보유
* 교육/성장/습관 관리 관심 높음

## 아이 (Secondary User)

* 6~10세
* 간단한 게임 및 UI 사용 가능

---

# 3. 핵심 가치 (Value Proposition)

1. **Today 중심**

   * 매일 접속해야 하는 이유 제공

2. **데이터 기반 육아**

   * 감이 아닌 데이터 기반 판단

3. **AI 코칭**

   * 부모 의사결정 지원

4. **놀이 + 학습 결합**

   * 아이 참여 유도

---

# 4. 제품 구조 (Information Architecture)

## 메인 탭 구조

1. Home (Today)
2. Growth (성장)
3. Record (기록)
4. Play (놀이)
5. Insight (AI 리포트)

---

# 5. 기능 정의 (Functional Requirements)

---

# 5.1 Home (Today Dashboard)

## 목적

매일 접속을 유도하는 핵심 화면

---

## 기능 목록

### 1. 오늘 일정

* 학원/학교 일정 표시
* 종료 시간 및 남은 시간 표시

---

### 2. 환경 정보

* 날씨
* 미세먼지
* 급식 메뉴

---

### 3. 오늘의 독서

* 오늘 목표 시간/권수
* 최근 읽은 책

---

### 4. AI 오늘 코칭

* 하루 행동 가이드
* 부모 대화 추천

---

## 데이터 요구사항

* schedule
* weather API
* meal API
* reading_logs

---

---

### 2. 기억력 게임

* 카드 매칭

---

### 3. 독서 기반 게임

* 단어 퀴즈
* 내용 기억 게임

---

* 게임 결과 → 성장 분석 반영
