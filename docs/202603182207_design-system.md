# Yuno920 디자인 시스템

> AI 기반 아이 성장·육아 통합 플랫폼 — 모바일 우선 디자인

---

## 1. 디자인 원칙

| 원칙 | 설명 |
|------|------|
| **따뜻함** | 육아·성장 서비스답게 부드럽고 따뜻한 톤 |
| **단순함** | 한 손으로 기록 가능, 복잡한 메뉴 최소화 |
| **일관성** | 같은 동작 = 같은 UI = 같은 위치 |
| **접근성** | 조부모도 읽기 쉬운 크기, 충분한 색상 대비 |
| **모바일 우선** | 모든 화면을 375px 기준으로 먼저 설계 |

## 3. 타이포그래피

### 3-1. 폰트

| 용도 | 폰트 | 비고 |
|------|------|------|
| 한글 본문 | **Pretendard** | Google Fonts CDN 제공 |
| 영문/숫자 | **Geist** | Next.js 기본 포함 |
| 코드/수치 | **Geist Mono** | 게임 점수, 통계 표시 |

### 3-2. 크기 체계

| 레벨 | 크기 | 굵기 | 줄높이 | Tailwind | 용도 |
|------|------|------|--------|----------|------|
| Display | 32px | Bold (700) | 40px | `text-3xl font-bold` | 랜딩 헤드라인 |
| H1 | 24px | Bold (700) | 32px | `text-2xl font-bold` | 페이지 제목 |
| H2 | 20px | SemiBold (600) | 28px | `text-xl font-semibold` | 섹션 제목 |
| H3 | 18px | Medium (500) | 26px | `text-lg font-medium` | 카드 제목 |
| Body L | 16px | Regular (400) | 24px | `text-base` | 본문 (기본) |
| Body S | 14px | Regular (400) | 20px | `text-sm` | 보조 텍스트 |
| Caption | 12px | Medium (500) | 16px | `text-xs font-medium` | 라벨, 태그, 타임스탬프 |

## 4. 간격 시스템 (Spacing)

### 4-1. 기본 단위: 4px

|------|-----|----------|------|

| lg | 20px | `5` | 카드 간 간격 |
| xl | 24px | `6` | 섹션 내 그룹 간 |
| 2xl | 32px | `8` | 섹션 간 간격 |
| 3xl | 48px | `12` | 페이지 주요 구간 |
