# Yuno920 와이어프레임 v2

> 모바일 우선 (375px 기준) — 피드백 반영 개선본
> v1 대비 주요 변경: 정보 우선순위 재배치, 탭별 헤더 분화, 기록 2모드, Growth 입력 동선, Play 보상 구조, Insight 행동 카드화

---

## 변경 이력

| 항목 | v1 | v2 변경 |
|------|-----|---------|
| Home | 날씨·일정·독서·급식 동일 비중 | 일정 중심 → 코칭 → 참고정보 순 위계 부여 |
| Header | 공통 1종 | 탭별 목적에 맞는 헤더 패턴 5종 정의 |
| Record 작성 | 단일 입력 폼 | 빠른 기록 / 상세 기록 2모드 분리 |
| Growth | 차트 보기 전용 | 각 카드에 +입력 동선 명시 |
| Play | 게임 허브 | 오늘의 미션 + 주간 배지 + 레벨 시스템 추가 |
| Insight | 텍스트 중심 | 행동 카드 4종(요약/코칭/기질/추천액션) 분리 |
| 설정 진입 | 미정의 | Home 헤더 아바타 → 설정 경로 명시 |
| MVP 단계 | 미구분 | 화면별 MVP / 2차 / 3차 태그 부여 |

---

## MVP 단계 분류

### MVP (1차 출시)
- 온보딩 / 로그인 / 아이 프로필 등록
- Home (Today Dashboard)
- Record 목록 + 작성 (빠른/상세)

### 2차
- Growth (성장)
- Insight (AI 리포트)

### 3차
- Play (놀이)
- 설정 고도화

|-----|------|------|------|------|

### Home 정보 위계 원칙

| 순서 | 영역 | 목적 | 크기 비중 |
|------|------|------|----------|

## 5. Growth (성장) `2차`

> **v2 변경**: 각 카드에 +입력 버튼을 명시하여 "보기 화면"과 "입력 동선"의 역할 분리.
> Growth는 보는 화면이고, 입력은 각 카드 우측 상단 +버튼 또는 하단 FAB에서 타입별로 분기.

```
┌─────────────────────────────┐
│  ←    성장 기록     기간▼    │
├─────────────────────────────┤
│                             │
│  ── 신체 성장 ─────────────  │
│  ┌───────────────────────┐  │
│  │  키: 128.5cm  몸무게: 27kg│  │
│  │                  [+ 입력]│  │
│  │  ┌─────────────────┐   │  │
│  │  │   [꺾은선 그래프]  │   │  │
│  │  │   키/몸무게 추이   │   │  │
│  │  └─────────────────┘   │  │
│  │  최근입력 3/15          │  │
│  └───────────────────────┘  │
│                             │
│  ── 6각형 인재 분석 ────────  │
│  ┌───────────────────────┐  │
│  │                  [상세▸]│  │
│  │    [레이더 차트 6각형]   │  │
│  │    학습 / 신체 / 사회성  │  │
│  │    감정 / 창의성 / 습관  │  │
│  │                        │  │
│  │  AI: "창의성 영역이      │  │
│  │  빠르게 성장하고 있어요"  │  │
│  └───────────────────────┘  │
│                             │
│  ── 학습 지표 ─────────────  │
│  ┌───────────────────────┐  │
│  │  SR 점수 추이   [+ 입력] │  │
│  │  [막대 그래프]            │  │
│  │  최근입력 3/17           │  │
│  └───────────────────────┘  │
│                             │
│  ── 독서 성장 ─────────────  │
│  ┌───────────────────────┐  │
│  │  이번달: 8/10권  [목표수정]│  │
│  │  [분야별 도넛 차트]       │  │
│  │  과학 35% 동화 30% ...   │  │
│  └───────────────────────┘  │
│                             │
├─────────────────────────────┤
│  Home Growth Record Play AI │
└─────────────────────────────┘
```

### Growth 입력 동선 정의

| 데이터 | 입력 위치 | 방식 |
|--------|----------|------|
| 키/몸무게 | 신체 성장 카드 [+ 입력] | 바텀시트 — 키, 몸무게, 날짜 |
| SR 점수 | 학습 지표 카드 [+ 입력] | 바텀시트 — 점수, 시험명, 날짜 |
| 독서 목표 | 독서 성장 카드 [목표수정] | 인라인 수정 |
| 독서 기록 | Record 작성 > 독서 카테고리 | Record에서 자동 연동 |
| 6각형 점수 | 자동 산출 (기록 기반 AI) | 수동 입력 없음 |

### 기록 모드 비교

| 항목 | 빠른 기록 | 상세 기록 |
|------|----------|----------|
| 기분 선택 | O | O |
| 한 줄 텍스트 | O (필수) | O (자유 분량) |

| 날짜 변경 | 오늘 고정 | 변경 가능 |
| 예상 소요 | ~10초 | ~1분 |

```

│          놀이터       [🏆]   │
├─────────────────────────────┤
│                             │
```
