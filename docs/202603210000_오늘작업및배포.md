## 1. 계산 게임 개발 (Play)

### 1-1. 신규 페이지
| 항목 | 내용 |
|------|------|
| 경로 | `/play/calculation` |
| 접근 | Play 메인 → 계산 게임 카드 클릭 |

### 1-2. 게임 규칙
- **연산**: 더하기(+), 빼기(-), 곱하기(×), 나누기(÷)
- **답 입력 방식**: 4개 보기 중 클릭으로 선택 (직접 입력 → 객관식 변경)
- **레벨업**: 5문제 연속 정답 시 다음 레벨로 진행
- **콤보**: 틀리면 리셋

### 1-3. 난이도 레벨 (9단계)
| LV | 이름 | 연산 |
|----|------|------|
| 1 | 더하기 기초 | 1~9 덧셈 |
| 2 | 더하기 도전 | 1~20 덧셈 |
| 3 | 빼기 기초 | 1~20 뺄셈 |
| 4 | 더하기 & 빼기 | 덧셈·뺄셈 혼합 |
| 5 | 곱하기 기초 | 2~9 곱셈 |
| 6 | 곱하기 도전 | 2~12 곱셈 |
| 7 | 나누기 기초 | 2~9 나눗셈 |
| 8 | 나누기 도전 | 2~12 나눗셈 |
| 9 | 사칙연산 마스터 | 사칙연산 혼합 |

### 1-4. UX
- 정답 시 초록색 피드백, 오답 시 빨간색 피드백
- 보기는 정답 주변의 비슷한 숫자로 구성 (혼선 유발)
- 선택 후 다른 보기 클릭 불가
