## 1. 백엔드 — Sprint 1 보완 + Sprint 2 신규

### 1-1. AI Coaching 실제 연동 (`/api/ai/coaching`)
- `OpenAIService.generate_coaching()` 메서드 구현
- 아이 이름, 나이, 오늘 일정, 최근 기록, 최근 독서 데이터를 Supabase에서 조회
- OpenAI `gpt-4o-mini`에 전달하여 맞춤 코칭 생성
- JSON 형태 응답 (coaching + tips[])

### 1-2. Weather 실제 연동 (`/api/external/weather`)
- Open-Meteo API (무료, API 키 불필요) 연동
- 위도/경도를 Query Parameter로 전달 (기본: 서울 37.5665, 126.9780)
- WMO 날씨 코드 → 한국어 설명 + 이모지 변환
- 에러 시 fallback 응답 제공

### 1-3. 급식 API 연동 (`/api/external/meal`)
- NEIS 급식 API 연동
- 학교 코드 + 날짜로 메뉴 조회
- `<br/>` 파싱하여 메뉴 리스트 반환

### 1-4. Daily Summary 실제 연동 (`/api/ai/daily-summary`)
- 해당 날짜의 기록 + 독서 로그를 조회
- OpenAI로 200자 이내 하루 요약 생성
- `ai_reports` 테이블에 결과 저장

### 1-5. 기록 자동 태깅 (`/api/ai/auto-tag`)
- `OpenAIService.auto_tag_record()` 메서드 구현
- 기록 텍스트를 분석하여 7가지 카테고리 중 1~3개 자동 추출
- 추출된 태그를 `records.categories` 컬럼에 업데이트

### 1-6. Supabase Service 확장 (`supabase_service.py`)
- `SupabaseService` 클래스 신규 생성
- `get_child`, `get_today_schedules`, `get_recent_records` 등 10개 메서드
- DB 조회/저장 로직을 서비스 계층으로 분리

### 1-7. 의존성 추가
- `httpx>=0.27.0` (비동기 HTTP 클라이언트)
