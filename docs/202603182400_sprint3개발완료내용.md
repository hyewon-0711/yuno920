## 1. 요구사항 반영

- **각 영역 입력 가능**: 신체(키/몸무게), 학습(점수), 6각형(직접 입력 또는 AI)
- **트래킹**: 시간별 추이 차트 (Recharts)
- **AI 분석**: 6각형 자동 산출, 부족 영역 개선 조언

## 3. 백엔드

### 3-1. AI 6각형 자동 산출 (`POST /api/ai/hexagon/calculate`)
- `OpenAIService.calculate_hexagon()`: 기록·독서·성장 지표·일정 기반 6영역 점수 산출
- `SupabaseService.save_hexagon_scores()`: 결과를 `hexagon_scores`에 저장
- 6영역: learning, physical, social, emotion, creativity, habit (0~100)

### 3-2. 성장 조언 개선 (`POST /api/ai/growth-advice`)
- 응답에서 번호 목록(1. 2. 3.)을 파싱해 `recommendations` 배열로 반환

### 3-3. SupabaseService 확장
- `get_growth_metrics(child_id, months)`
- `get_reading_logs_for_period(child_id, months)`
- `save_hexagon_scores(...)`

---

## 4. 프론트엔드

### 4-1. GROW-01: 신체 성장
| 항목 | 내용 |
|------|------|
| 입력 | 키(cm), 몸무게(kg), 측정일, 메모 |
| 차트 | 이중 Y축 꺾은선 (키/몸무게) |
| Empty | "아이의 키와 몸무게를 기록해보세요" |

### 4-2. GROW-02: 학습 지표
| 항목 | 내용 |
|------|------|
| 입력 | 학습 점수(0~100), 측정일, 메모 |
| 차트 | 세로 막대 그래프 |
| 레벨 | 우수(80+), 양호(60+), 보통(40+), 보완 필요 |

### 4-3. GROW-03: 6각형 인재 분석
| 항목 | 내용 |
|------|------|
| AI 분석 | 버튼 클릭 시 백엔드 `/hexagon/calculate` 호출 → 자동 산출·저장 |
| 직접 입력 | 6영역 슬라이더(0~100) → 저장 |
| 차트 | 레이더 차트 (Recharts RadarChart) |

### 4-4. GROW-04: 독서 성장
| 항목 | 내용 |
|------|------|
| 데이터 | `reading_logs` 월별 집계 |
| 차트 | 월별 읽은 시간(분) + 책 권수 막대 그래프 |

### 4-5. GROW-05: AI 성장 조언
| 항목 | 내용 |
|------|------|
| 입력 | 6각형 점수 50 미만 영역을 `weak_areas`로 전달 |
| 출력 | 종합 조언 + 구체 활동 3~5개 |
